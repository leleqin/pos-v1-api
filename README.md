# pos-v1-api
### 需求描述
1. 使用express框架实现pos机（参考前面的pos机命令行任务卡需求）的基本api
    - GET    /items           获取全部商品
    - GET    /items/:id       获取一个商品
    - POST   /items           增加一个商品
    - DELETE /items/:id       删除商品
    - PUT    /items/:id       更新数据

2. 实现处理商品条码数组，返回购物清单的api
清单内容(例样):
```js
***<没钱赚商店>收据*** 
名称:可口可乐，数量:3瓶，单价:3.00(元)，小计:6.00(元)
名称:羽毛球，数量:5个，单价:1.00(元)，小计:4.00(元)
名称:苹果，数 :2 ，单价:5.50(元)， 计:11.00(元) 
----------------------
总计:21.00(元)
节省:4.00(元)
**********************

```
